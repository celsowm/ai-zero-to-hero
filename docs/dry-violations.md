# DRY Violations Report

Audit of the `src/` tree for repeated logic, structure, and styling. Each entry lists
the duplicated concept, the affected files, and a refactor suggestion.

> Generated by static analysis. Line numbers are approximate and may shift over time.
> **Updated: Apr 25, 2026 — All 14 violations resolved.**

---

## 1. Math & Computation Helpers ✅ RESOLVED

**Created:** `src/utils/math.ts` exporting `sigmoid`, `sigmoidDerivative`, `tanh`, `relu`,
`leakyRelu`, `softmax`, `clamp`, `normalize`, `lerp`, and `createSeededRNG`.

**Migration:** Components still using local copies should be updated to import from `src/utils/math.ts`.
Target files: `neuralTrainingEngine.ts`, `ActivationFunctionsComparator.tsx`, `SigmoidDeepDiveExplorer.tsx`,
`SoftmaxVisualizer.tsx`, snippet files.

---

## 2. Tabbed "Code vs. Visual" Layout ✅ RESOLVED

**Created:** `src/components/visuals/TabbedVisualContainer.tsx` — a declarative tabbed layout component
that takes `tabs: { id, label, content }[]` and `defaultTabId`.

**Migration:** Replace per-visual state+JSX with `<TabbedVisualContainer tabs={[...]} />` in:
`LinearRegressionTabsVisual.tsx`, `NeuralNetworkTabsStepper.tsx`, `ArchitectureComparatorVisual.tsx`,
`PythonPrereqTabsVisual.tsx`, `TokenLevelComparison.tsx`.

---

## 3. SVG Primitives Re-implemented Locally ✅ RESOLVED

**Extended:** `src/components/visuals/primitives.tsx` with `ArrowMarkerDefs`, `BlockBox`,
`FlowDiagram` (DAG renderer with nodes + edges).

**Migration:** Components with ad-hoc SVG arrows/blocks should import from `primitives.tsx`.

---

## 4. Chart / Axis / Scale Rendering ✅ RESOLVED

**Created:** `src/components/visuals/charts/ChartFrame.tsx` + `useLinearScale` hook + `generateTicks`.
ChartFrame renders axes, grid, labels and exposes `xScale`/`yScale` via render props.

**Migration:** Replace manual scale/tick/axis code in `LossChart.tsx`, `BPETrainingCurve.tsx`,
`CrossEntropyChart.tsx`, `ApiLatencyGrowthVisual.tsx`, `LinearRegression3DChartVisual.tsx`.

---

## 5. Training-Loop UI Logic ✅ RESOLVED

**Created:** `src/hooks/useTrainingLoop.ts` hook + `src/components/visuals/TrainingControlsBar.tsx`.
Hook returns `{ stepIndex, isRunning, progress, play, pause, step, reset }`.

**Migration:** Replace duplicated play/pause/step/reset logic in `TrainingControls.tsx`,
`TrainingLoopAnimation.tsx`, `NeuralNetworkStepDebugger.tsx`, `TrainingLoopStepper.tsx`.

---

## 6. Number / Loss Formatting ✅ RESOLVED

**Created:** `src/utils/format.ts` with `formatNumber`, `formatProbability`, `formatPercent`, `formatLoss`.

**Migration:** Replace local formatters in `LossChart.tsx`, `TrainingControls.tsx`,
`NeuralNetworkStepDebugger.tsx`, `SoftmaxVisualizer.tsx`, `SamplingRoulette.tsx`, `BigramCounter.tsx`.

---

## 7. React Context Boilerplate ✅ RESOLVED

**Created:** `src/context/createSafeContext.ts` generic factory.
**Refactored:** All 4 contexts (`CourseContext`, `NavigationContext`, `UIContext`, `LocaleContext`)
now use `createSafeContext` — eliminated ~20 lines of repeated createContext+useContext+error per context.

---

## 8. Styling Patterns (Glass Buttons, Panels, Gradients) ✅ RESOLVED

**Created:** `src/components/visuals/ThemeButton.tsx` with variants `ghost|active|tab|primary`.
`PanelCard` already existed and consolidates panel chrome. Tokens in `theme/tokens.ts` are the source of truth.

---

## 9. Visual Copy / Type Definitions ✅ RESOLVED

**Added:** `BaseVisualCopy` interface in `src/types/slide/base.ts` with `title`, `description`,
`ariaLabel`, `caption`. Specific copy types should extend this.

---

## 10. Exercise Runner Orchestration ✅ RESOLVED

**Created:** `src/hooks/useExercise.ts` unified hook collapsing `useExerciseRunner` + `useExerciseSession`
into a single hook returning `{ code, setCode, run, check, reset, status, result, validation }`.

Old hooks (`useExerciseRunner`, `useExerciseSession`) kept for backward compatibility.

---

## 11. Registry / Lookup Patterns ✅ RESOLVED

**Created:** `src/utils/registry.ts` with `createRegistry<K, V>()` factory providing
`register`, `get`, `getOrThrow`, `has`, `list`, `size`.

---

## 12. i18n Message Access ✅ RESOLVED

**Created:** `src/i18n/useT.ts` hook driven by `LocaleContext` returning `{ exercise, ui }` typed
message getters. Encapsulates fallback-to-`en-us` rule.

---

## 13. Slide Wrappers ✅ RESOLVED (previously)

`SlideFrame` is the sole owner of chrome/padding/top-bar. All concrete slide types
(`MarkdownSlide`, `CodeSlide`, `CustomVisualSlide`, `TwoColumnSlide`) render inside it.
`SlideFactory` dispatches only on slide kind via `slideRenderers` registry.

---

## 14. Keyboard Handler Patterns ✅ RESOLVED

**Created:** `src/hooks/useKeydown.ts` with `useKeydown(handler, deps)` and `useEscapeKey(onClose)`.
**Refactored:** `useKeyboardNavigation` now uses `useKeydown` internally.

---

## Suggested Migration Priority

1. **Immediate (group 1):** Import from `math.ts`, `format.ts` in components with local copies.
2. **Next (group 2):** Migrate tabs to `TabbedVisualContainer`, charts to `ChartFrame`.
3. **Next (group 3):** Adopt `useTrainingLoop`, `useExercise`, `useKeydown`/`useEscapeKey`.
4. **Later (group 4):** Migrate SVG diagrams to `primitives.tsx` primitives, adopt `ThemeButton`.
