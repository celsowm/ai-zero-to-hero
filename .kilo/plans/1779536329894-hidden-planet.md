# Plan: Transform right columns into abas (tabs) with didatic tensor visualization

## Goal

- **Tab 1** keeps the existing snippet content (code)

## Slides affected

### A. `neural-network-pytorch-tensors` (B/F/H/O, patient tabular example)

Currently:
- `rightBody` â†’ `snippet:pytorch-lm/patient-shape-contract` (single Python snippet)
- Tab 1: same code snippet
- Tab 2: **New `SequentialShapeFlow` visual** showing:
  - The actual `model = nn.Sequential(nn.Linear(4,3), nn.Sigmoid(), nn.Linear(3,1), nn.Sigmoid())` architecture
  - Real tensor shapes flowing: `X (6,4)` â†’ `hidden (6,3)` â†’ `y_hat (6,1)`

### B. `pytorch-shapes-language-modeling` (B/T/C/V, language modeling)

Currently:
- `rightBody` â†’ `snippet:pytorch-lm/tensor-primer` (single Python snippet)

- Tab 2: **New `LanguageModelingShapeFlow` visual** showing:
  - The three contracts visually: `token_ids (B,T)` â†’ `hidden_states (B,T,C)` â†’ `output_scores (B,T,V)`
  - Real shapes: `(2,4)` â†’ `(2,4,8)` â†’ `(2,4,50)`

  - Indexed access: `token_ids[0,1]` (scalar), `hidden_states[0,1]` (vector of length C), `output_scores[0,1]` (vector of length V)

### 3. New visual components

**a) `SequentialShapeFlow.tsx`**
- Follows the pattern of `Tensor3DExplorer.tsx` (uses `TabsBar` + `TabbedPanelSurface`)
- Tab 0 ("Code"): renders the existing snippet via `CodeBlock` with `sourceRef` pointing to the patient-shape-contract snippet
- Tab 1 ("Flow"): renders an SVG/HTML visual showing:
  - A horizontal pipeline diagram with stages: Input â†’ Linear1 â†’ Sigmoid â†’ Linear2 â†’ Sigmoid â†’ Output
  - Each stage labeled with its shape contract: `(B,F)`, `(B,H)`, `(B,H)`, `(B,H)`, `(B,O)`, `(B,O)`
  - Concrete shape values: `(6,4)`, `(6,3)`, `(6,3)`, `(6,3)`, `(6,1)`, `(6,1)`
  - A concrete tensor preview section showing small sections of X, hidden, y_hat with actual values

**b) `LanguageModelingShapeFlow.tsx`**
- Same pattern: `TabsBar` + `TabbedPanelSurface`
- Tab 0 ("Code"): renders the existing snippet via `CodeBlock` with `sourceRef` pointing to the tensor-primer snippet
- Tab 1 ("Flow"): renders a visual showing:
  - Three stacked horizontal "tracks" representing `token_ids`, `hidden_states`, `output_scores`
  - Each track shows the shape for B=2, T=4, and the respective third dimension (C=8, V=50)
  - Visual highlight of `[0, 1]` position across all three levels
  - Color-coded dimensions
